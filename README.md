# Resume Tailor - ATS Optimizer

This tool helps you tailor your resume to specific job descriptions using Claude AI, maximizing ATS keyword matching through a simple two-step workflow.

## Quick Start

### Prerequisites

- **Python 3.10+** with pip
- MS Word or LibreOffice (for PDF conversion) - optional

### Installation

```bash
pip install -r requirements.txt
```

### Prepare Your Template

If you haven't already, run once to create your ATS-friendly template:

```bash
python rebuild_cv.py
```

This creates `AamirAli_Resume_ATS.docx` from your original resume.

## Two-Step Workflow

### Step 1: Generate Prompt

Generate a tailored prompt from a job description:

```bash
python tailor_resume.py --mode prep --jd job_description.txt
```

Or manually paste the JD when prompted:

```bash
python tailor_resume.py --mode prep
```

Copy the prompt from `prompt.txt` and paste it into **Claude's chat interface** at https://claude.ai.

### Step 2: Apply Claude's Response

After Claude responds with the JSON:

1. Copy the complete JSON response
2. Save it as `response.json` in the project folder
3. Run:

```bash
python tailor_resume.py --mode apply
```

This generates:
- A tailored `.docx` file in `generated_resumes/`
- A PDF version (if MS Word or LibreOffice is available)

### Optional: Check ATS Compliance

Verify your generated resume is ATS-friendly:

```bash
python tailor_resume.py --mode check --doc generated_resumes/Your_Company.docx
```

## What Gets Tailored

- **Skills** - adds missing keywords from JD (from predefined adjacent skills)
- **Experience bullets** - rewrites each bullet to include JD keywords (~200 chars, STAR format)
- **ATS compliance** - checks for tables, text boxes, multi-column layouts

All formatting (bold markers, bullet characters) is preserved.

## Tips for Best Results

1. **Quality JDs matter** - use the full job description, not just bullet points
2. **Check ATS score** - The JSON response includes `ats_match_percentage`. Aim for 80%+
3. **Review before applying** - Open the generated docx and verify it looks good
4. **Iterate** - If ATS score is low, consider:
   - Adding missing keywords to your CV_CONTENT permanently
   - Manually tweaking the generated response.json and re-running `--mode apply`

## Troubleshooting

**"Template not found: AamirAli_Resume_ATS.docx"**
- Run `python rebuild_cv.py` first to create it from your original resume
- Or create a symlink to your existing template

**PDF conversion fails**
- Install MS Word (docx2pdf) or LibreOffice
- Or use online converters: upload .docx to Google Docs → Download as PDF
- The .docx is still generated even if PDF fails

**JSON parse errors**
- Make sure you're copying Claude's full JSON response (no extra text)
- Claude sometimes adds markdown fences - the script strips them automatically
- If errors persist, check the raw response saved in `debug_claude_response.txt`

## Advanced Usage

### Custom prompt engineering
Edit `build_prompt()` in `tailor_resume.py` to change how prompts are constructed.

### Batch processing (manual)
If you want to process multiple JDs:

```bash
for jd in jd/*.txt; do
    python tailor_resume.py --mode prep --jd "$jd"
    # Manually copy prompt.txt to Claude, save response as response.json
    # Then run: python tailor_resume.py --mode apply
done
```

## Git Workflow

This repo excludes:
- Generated resumes (`*.docx`, `generated_resumes/`)
- Temporary files (`prompt.txt`, `response.json`, `*.log`, `__pycache__/, .env`)

**Tracked files** (safe to commit):
- `tailor_resume.py`
- `rebuild_cv.py`
- `requirements.txt`
- `jd/` folder (your job descriptions)
- `.gitignore`

**Never commit:**
- `AamirAli_Resume_ATS.docx` (contains personal info)
- `response.json` (may contain PII from JDs)
